# Arguments to main (*argc* and *argv*)

This project covers the two arguments to the main function and how they can be
used to manipulate command line arguments. Below are a few of the programs and
their descriptions.

**0-whatsmyname.c:** a program that prints its name, followed by a new line.

**1-args.c:** a program that prints the number of arguments passed into it.

**2-args.c:** prints all arguments it receives, one each per line.
